취약점 소개
--------------------------
외부로부터 입력된 값을 검증하지 않고 입·출력 함수의 포맷 문자로 그대로 사용하는 경우 발생하는 취약점

점검 대상
--------------------------
- 웹 애플리케이션 소스 코드(C언어 기반으로 짜여진 경우 진단)
- 웹 기반 C/S 프로그램

진단 방법
--------------------------
- 포맷 스트링을 이용하는 함수를 사용하는 경우 인자로 포맷 스트링이 포함되어 있는지 확인
- 외부 입력 값이 포맷 스트링 생성에 사용되는지 확인
